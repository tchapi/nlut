NLU Transcoder
---

For **PHP7**.

This little PHP CLI tool transcodes Alexa Skill / Wit.ai / DialogFlow backup files in another format for ease of switching to another platform if needed.

### Installation

_No steps are necessary prior to using the tool._

### Usage

    ./nlut.php --help
